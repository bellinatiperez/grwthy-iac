# Quickstart

## 1. Create an Account

First, create a WhatsApp Business account using the master API key:

```bash
curl -X POST https://your-domain.com/whatsapp/accounts \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: YOUR_MASTER_API_KEY" \
  -d '{
    "name": "My Business",
    "phoneNumberId": "YOUR_PHONE_NUMBER_ID",
    "businessAccountId": "YOUR_BUSINESS_ACCOUNT_ID",
    "accessToken": "YOUR_META_ACCESS_TOKEN",
    "appSecret": "YOUR_META_APP_SECRET",
    "webhookVerifyToken": "my-custom-verify-token"
  }'
```

The response includes an `apiKey` — **save it**, as it won't be shown again.

## 2. Send a Text Message

Use your account's API key to send a message:

```bash
curl -X POST https://your-domain.com/whatsapp/messages/text \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: YOUR_ACCOUNT_API_KEY" \
  -d '{
    "to": "5511999999999",
    "body": "Hello from Grwthy!"
  }'
```

## 3. Send a Template Message

Templates must be pre-approved in your Meta Business account:

```bash
curl -X POST https://your-domain.com/whatsapp/messages/template \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: YOUR_ACCOUNT_API_KEY" \
  -d '{
    "to": "5511999999999",
    "templateName": "hello_world",
    "languageCode": "en_US"
  }'
```

## 4. Configure Webhook

In your Meta App Dashboard, set the webhook URL to:

```
https://your-domain.com/whatsapp/webhook
```

Use the `webhookVerifyToken` you set when creating the account.
